# Trello CLI Docs

This folder tracks the next implementation slices for `TheNerdCollective.Cli.Trello`.

Current MVP covers:

- list card checklists
- ensure checklist exists
- add checklist item
- set checklist item state by id
- add comment to card

The documents here are intentionally short and execution-oriented so future work can be picked up in slices.

## Files

- `next-slices.md` — prioritized implementation backlog
- `ideas.md` — higher-value follow-up suggestions and longer-range improvements
